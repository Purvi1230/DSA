Description - Factorization
Print all factors of a given number n in ascending order.

Input Format
The only line of input contains an integer n.

Output Format
Print the answer on a new line.
Constraints
1 ≤ n ≤ 1012

Sample Input 1
12

Sample Output 1
1 2 3 4 6 12

Sample Input 2
9

Sample Output 2
1 3 9
